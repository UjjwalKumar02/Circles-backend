# Circles-backend

Backend of [Circles](https://github.com/UjjwalKumar02/Circles) project

## Tech stack:

- **Framework** : Express.js
- **Language** : TypeScript
- **WebSocket** : WS
- **Database** : PostgreSQL
- **ORM** : Prisma
- **Authentication** : Google oAuth with JWT (JsonWebToken)
